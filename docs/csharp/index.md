# C# 基礎

C# プログラミングの基本的な概念をゼロから学びます。

## このセクションの内容

| # | トピック | 概要 |
|---|---|---|
| 1 | 変数とデータ型（準備中） | 値を格納する変数と基本的な型 |
| 2 | 条件分岐（準備中） | if/else, switch による分岐処理 |
| 3 | ループ（準備中） | for, while による繰り返し処理 |
| 4 | メソッド（準備中） | 処理をまとめて再利用する |
| 5 | クラスとオブジェクト（準備中） | オブジェクト指向の基礎 |

## 前提知識

このセクションはプログラミング未経験の方を対象としています。特別な前提知識は不要です。

## 学習目標

- C# の基本的な文法を理解できる
- 簡単なプログラムを自分で書けるようになる
- Unity スクリプトを読んで理解できる基礎力をつける
